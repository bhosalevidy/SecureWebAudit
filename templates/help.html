WebAudit Help & Docs

Overview

WebAudit is an automated tool that performs functional checks on websites and presents a visual dashboard of results.

  • Checks website title, meta description, H1 tags, links, images.
  • Verifies CTA buttons, search input, navigation, and footer links.
  • Generates pass/fail dashboards with downloadable reports.
  • Simple, fast, and user-friendly interface.

How to Use

  1. Go to the Input Page.
  2. Enter the website URL you want to test.
  3. Click Run Test.
  4. Wait for the loader messages while tests run.
  5. Review the dashboard results showing summary cards, charts, and a detailed report.
  6. Download the report as CSV if needed.

Understanding Results

Summary Cards: Total Tests, Passed / Failed, and Success Rate.

Detailed Report: One-line description for each test result. CSV download available.

Charts: Donut chart (pass/fail ratio), Bar chart (distribution of results).

FAQ

Currently, WebAudit supports one URL at a time.
Any publicly accessible website. Login-protected pages are not supported. You can safely test your own websites (personal projects, college apps) and apps running on localhost or internal networks.
Usually under a minute for standard websites. Large sites may take longer.

Troubleshooting

  • Loader stuck: Check internet connection and website availability.
  • No results: Ensure the URL is correct.
  • Errors: Ensure browser and driver are properly configured for Selenium.
